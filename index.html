Операторы JS

Что бы не забыть синтаксис добавил сюда чуток JS

Применение арифметических операторов

  • = is x = y Example: x = y
  • += is x = x + y Example: x += y
  • -= is x = x - y Example: x -= y
  • *= is x = x * y Example: x *= y
  • /= is x = x / y Example: x /= y
  • %= is x = x % y Example: x %= y
Логические операторы!
  • && Логическое и (and) A && B
  • || Логическое или (or) A || B
  • ! Логическое не (not) !( A && B)
